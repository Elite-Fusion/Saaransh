Our project is **not a normal crime management system**. It is an **AI-powered Crime Intelligence Platform** for the Karnataka Police.

The frontend should feel like software used inside a **Police Control Room**, not a regular website.

---

# User Roles

Only **3 types of users** exist.

## 1. Police Station

Can:

* Register FIR
* Upload evidence
* View cases
* View AI suggestions
* Search previous cases

Cannot:

* Edit database records directly
* Manage users

---

## 2. Database Handler

Can:

* Verify imported data
* Manage master records
* Update crime categories
* Resolve duplicates
* Manage data quality

Cannot:

* Register FIR

---

## 3. Control Room

Highest privilege.

Can:

* Dashboard
* AI Investigation
* Crime Analytics
* Heat Maps
* Cross District Linking
* Risk Prediction
* Reports
